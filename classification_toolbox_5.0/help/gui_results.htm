Classification performances

It is possible to analyse the classification performances of the calculated model by clicking "results->classification results". The "view classification results" form will appear:


On the left of the form, error rate, non-error rate, specificity, sensitivity, precision and the ratio of not assigned samples (when higher than 0) are shown both in fitting (all the samples used to build the model) and validation (look here for further information on these classification parameters).
The "view confusion matrix" button shows the confusion matrix. The "view calculated class" shows the vector with the class assignment of each sample [n x 1]. Confusion matrix and calculated/predicted classes can be displayed both on the fitting and validation results.

Class boundaries

It is possible to analyse class boundaries only when the dataset has 2-dimensions or when models are calculated on two principal components / latent variables. In this case, the "class boundaries" menu is active; it opens a 2 dimensional plot with training samples coloured on the basis of their experimental class and the class boundaries.

Discriminant Analysis results

Canonical variables associated to Linear Discriminant Analysis can be analysed by selecting "results->canonical variables". Note that if Quadratic Discriminant Analysis is calculated, this option is not available. If Discriminant Analysis is calculated on the PC scores (PCA-DA), the corresponding scores and loadings can be analysed ("results->PCADA scores and loadings"). A form will appear with score and loading plots. In this plot, samples are coloured on the basis of their experimental class. The user can modify the components to be analysed on the plots, while other statistics are available in the result form, such as Q residuals and T2 Hotelling. The score and loading plots can be exported as MATLAB figures. The "view sample" button enables the user to select a specific point in the score plot: profiles of raw and scaled data, Q residuals contributions and Hotelling T2 contributions of the selected sample will appear in a new plot.

Partial Least Square - Discriminant Analysis results

PLSDA results can be analysed by selecting "results->PLSDA scores and loadings". A form will appear with score and loading plots. In this plot, samples are coloured on the basis of their experimental class. The user can modify the components to be analysed on the plots, while other statistics are available in the result form, such as Q residuals, T2 Hotelling, leverages, y calculated/predicted classes, regression coefficients. The "view sample" button enables the user to select a specific point in the score plot: profiles of raw and scaled data, Q residuals contributions and Hotelling T2 contributions of the selected sample will appear in a new plot. All plots can be exported as MATLAB figures. The "results->ROC curves" option will open a plot with the ROC curve of each class.

The classification tree calculated by CART can be analysed by selecting "results->CART tree".

Potential Functions (Kernel Density Estimators) results

Potential Functions results can be analysed by selecting "results->PF". A form will appear with plots of potentials for each class, as well as the class boundaries projection when dealing with models calculated on 2D data (or on 2 Principal Components). In this plot, samples are coloured on the basis of their experimental class. The "view sample" button enables the user to select a specific point in the score plot: profiles of raw and scaled data of the selected sample will appear in a new plot.

[-> top]


Support Vector Machines (SVM) results

SVM results can be analysed by selecting "results->SVM". A form will appear with plots of alpha values and distances of samples from the class boundary. The class boundaries projection can be produced when dealing with models calculated on 2D data (or on 2 Principal Components). In this plot, samples are coloured on the basis of their experimental class. The "view sample" button enables the user to select a specific point in the score plot: profiles of raw and scaled data of the selected sample will appear in a new plot.

[-> top]


Soft Independent Modeling of Class Analogy (SIMCA) results

SIMCA results can be analysed by selecting "results->SIMCA scores and loadings". A form will appear with score and loading plots for each class PCA model. In this plot, samples are coloured on the basis of their experimental class. The user can modify the components to be analysed on the plots, while other statistics are available in the result form, such as Q residuals, T2 Hotelling and class distances. The "view sample" button enables the user to select a specific point in the score plot: profiles of raw and scaled data, Q residuals contributions and Hotelling T2 contributions of the selected sample will appear in a new plot. All plots can be exported as MATLAB figures.

[-> top]


Unequal class models (UNEQ) results

UNEQ results can be analysed by selecting "results->UNEQ scores and loadings". A form will appear with score and loading plots for each class PCA model. In this plot, samples are coloured on the basis of their experimental class. The user can modify the components to be analysed on the plots, while other statistics are available in the result form, such as normalised T2 Hotelling for each class model. The "view sample" button enables the user to select a specific point in the score plot: profiles of raw and scaled data, Q residuals contributions and Hotelling T2 contributions of the selected sample will appear in a new plot. All plots can be exported as MATLAB figures.
